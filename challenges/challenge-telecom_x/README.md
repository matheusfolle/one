# Análise de Churn de Clientes

## Sobre o projeto

Este projeto tem como objetivo analisar dados de clientes para entender o comportamento de evasão (**churn**). O churn ocorre quando um cliente cancela um serviço, algo que pode impactar diretamente a receita de uma empresa.

A análise foi realizada utilizando Python em um Jupyter Notebook, explorando os dados para identificar padrões que possam indicar quais fatores estão mais associados ao cancelamento de clientes.

## O que foi feito

Durante o projeto foram realizadas etapas de preparação e exploração dos dados. Primeiro os dados foram importados e analisados para entender sua estrutura. Em seguida foi feita a limpeza e tratamento das informações, incluindo verificação de valores ausentes e conversão de colunas para o formato numérico quando necessário.

Depois disso foi realizada uma análise exploratória utilizando estatísticas descritivas e visualizações gráficas. Foram analisadas variáveis categóricas, como tipo de contrato e método de pagamento, além de variáveis numéricas como tempo de contrato, valor mensal e valor total pago pelos clientes.

Essas análises permitiram identificar alguns padrões relacionados ao churn, como maior evasão entre clientes com contratos mensais e clientes com menor tempo de permanência no serviço.

## Tecnologias utilizadas

Python, Pandas, Matplotlib, Seaborn e Google Colab.
